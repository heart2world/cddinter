买不起的流量，那么1.7亿日活的小程序可以拯救

2018年03月05日189

从PC到APP再到微信，互联网的每波流量红利都会犒赏嗅觉敏感及执行力的强者。但是最终的胜利者，往往还是那些潜心沉淀自己核心价值的企业。

在移动社交流量时代，坐拥9.8亿月活用户的微信无疑是一个绕不过去的流量重镇。从去年Q4开始，随着社交支付的完善和小程序的更新迭代，微信生态的流量又形成了新的爆发点。在450多家经纬系公司中，使用小程序的超过30%，而在一年前这一数据低于10%。

为什么会有这种变化？小程序有多大的流量价值？是否能沉淀用户价值？我们做了一些研究，希望对互联网营销负责人或者希望通过小程序找到流量突破的创业者有帮助。以下，Enjoy：

一、为什么去年小程序Q4爆发了？

小程序正式上线是在2017年1月9日，但是初期各类企业的反应并不积极，大家都在观望究竟微信对小程序的支持力度有多大。随后的一年多时间里，小程序更新迭代了100多项能力，开放了近50多个入口，其中比较重要的是：与更多的公众号开放关联，降低开发门槛，上线“社交立减金”，提升群分享好友，打开支付的通道，增加找回的功能，提供更多的数据。

而最终随着春节前“跳一跳”这款游戏的出现，让小程序更加普及和深入人心。由于可以更便捷地获取，释放更强大的功能，链接更多的场景，持续的优化和迭代使得小程序在一年内上线数量达到70万，小程序的累积用户数超过4亿，全面覆盖200+细分行业。

二、小程序的流量特点

小程序定位于用完即走的轻应用，依托微信去中心化的理念，整体流量呈现出了新的特点：

1.强社交

微信的生态属于强关系链的私域流量，目前小程序的流量入口主要依赖分享 / 公众号 / 发现，并不是大流量的入口，但是由于便于转发好友和微信群，所以往往具有很强的社交裂变能力，通过社交机制，熟人背书，获得高转化裂变式的爆发。

2.人群下沉

从人群看，也是由于强社交的属性，小程序除了能触达主流的一二线城市人群，比较容易接触到不习惯使用APP的三四线城市人群以及中老年人群。

3.留存难

小程序的设计初衷是用完即走，用户决策链短，平均使用时长5分钟左右。如果说小程序的获客可以形成裂变降低成本，留存低于普遍水平——小程序7天的留存只有3.2%；而比较APP应用市场和PC端的获客，基本上是40%左右。也是基于此，很多企业把小程序定位于拉新利器。

三 、谁抢占了流量高地？

在去年上线的小程序中，阿拉丁统计平台对C端访问量和B端数量进行了对比，整体看零售 / 出行 / 工具 / 餐饮和生活服务是C端引入流量最高的应用场景，这里有个有趣的现象，B端数量多但C端访问量少的场景分别是：内容 / 泛娱乐 / 金融；而B端少C端多的则是：出行与交通 / 零售 / 餐饮。不可不说，后三种类型的玩家相对更能抢占到这波流量高地。从经纬系或其他优秀的玩家看：往往依托于高频的场景，刚需的工具，快捷的支付体验，更能吸引用户。

1.出行与交通

即时且高频的共享单车成为出行类最能享受小程序红利的代表。

2.零售

智慧零售类小程序通过铺设线下入口，凭借高效的引流和到店能力，运营支付卡包以及公众号等微信产品，在中低消费额度的购物商场和便利店等线下门店场景已经得到广泛应用；专注生鲜 / 时尚 / 母婴垂直电商类小程序，通过团拼和秒杀另外社交金立减的方式获得引爆。

3.餐饮

小程序在餐饮业的普及逐渐渗透到预约，到店点餐，外卖等场景，而小程序交单方便的会员使用体验和高效的会员管理体系，拉动了会员的持续消费。

四、沉淀用户更重要

其实你会发现，在使用的小程序过程中，最重要的不仅仅是获客，而是如何能够提升用户的迁移成本降低流失，增强用户的使用频次和粘性？这也是为什么，相对于初创的公司而言，成熟的公司却能更好的收割流量红利。

1.产品延展

如果你的产品功能点不足够有话题和传播性，实际上很难享受微信小程序的红利。而如何让用户持续为小程序买单，还需要充分挖掘产品的延展能力。

2.运营能力

如果你只懂得社交的方式获客，也必须要扎实地想好——如何能通过激励 / 好友关系或者等级等方式做好用户运营。此外，在渠道运营上，一是充分打通微信生态的的渠道很关键，结合公众号内容、微信群、朋友圈、推送消息传播形成联动；二是微信生态以外，也要想好如何与APP等渠道形成流量互补，最忌讳简单照搬和复制APP功能。

3.商业闭环打造

小程序能帮助你获得裂变的流量，但是也对核心能力是个考验。比如：社交电商必须要供应链和物流积累足够深，才能通过不断筛选商户以及规模效应保证商品低价，质量过关。

小程序在发展初期去中心化的概念引导市场，但实际上只是不主导流量的分发，却能帮助企业重新聚合成一个个独立又紧密的小型中心。2018年小程序势必会迎来下一个以重塑流量中心为主的爆发阶段。外部预计，小程序数量未来将突破300万个，而社交电商 / 餐饮 / 出行 / 游戏 / 内容媒体等企业都会更有机会。

但终归，我们还是想用你们常见的那句“自强则万强”——小程序的流量赋能价值，最终更考验的还是创业者产品 / 场景 / 商业闭环和内功的打造。
